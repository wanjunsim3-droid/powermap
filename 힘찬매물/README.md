# 🏠 힘찬 비공개 회원제 매물장 (Himchan Private Properties Board)

제공해주신 엑셀 데이터를 바탕으로 구축된 **비공개 회원제 매물 관리 웹 서비스**입니다. 
로그인 및 회원가입 후 관리자의 승인을 받은 사용자만 실시간 매물 검색 및 세부 정보 조회가 가능합니다.

---

## 🌟 주요 기능

### 1. 폐쇄형 회원제 시스템 (Authentication & Approval)
- **회원 가입 신청**: 아이디, 비밀번호, 실명, 연락처 정보를 바탕으로 가입을 신청합니다.
- **승인 대기 상태 (PENDING)**: 가입 직후에는 승인 대기 화면이 표시되며 모든 매물 API 접근이 철저히 차단됩니다.
- **관리자 승인 처리**: 관리자 계정(`admin`)으로 접속하여 가입 신청자를 **승인(APPROVED)** 혹은 **거절(REJECTED)**할 수 있습니다.

### 2. 프리미엄 매물 조회 대시보드
- **모던 화이트 UI**: Glassmorphism 디자인 시스템이 적용되어 가시성과 조작성이 뛰어난 깔끔한 레이아웃을 제공합니다.
- **통합 검색**: 주소, 상호, 상세 비고 란의 키워드를 기반으로 실시간 통합 검색을 수행합니다.
- **다차원 범위 필터**: 평수, 보증금, 임대료, 권리금의 **최소~최대 조건 범위**를 지정해 원하는 매물을 즉시 찾을 수 있습니다.
- **시트별 분류**: 엑셀 원본 시트(`26년`, `11월27일` 등) 기준으로 매물을 신속하게 분류 필터링합니다.
- **상세 뷰 모달**: 매물 카드를 클릭 시 팝업 모달이 노출되어 전체 상세 정보와 비고 내용을 상세히 표시합니다.

### 3. 네이버 지도 자동 연동 및 폴백 (Hyperlink Fallback)
- 엑셀 데이터에 하이퍼링크가 입력되어 있는 매물은 엑셀 내의 타겟 주소로 직접 연결됩니다.
- 하이퍼링크가 누락되었거나 단순 텍스트로 `'지도'`라고만 기입되어 있는 매물(**777건**)의 경우, 매물의 **지번 주소를 기반으로 네이버 지도 검색 URL을 동적으로 자동 생성**하여 지도 바로가기를 100% 정상 작동시킵니다.

### 4. 강력한 관리자 전용 기능 (Admin Console)
- **가입 회원 통제**: 회원가입 신청 목록 관리 및 승인/거절 상태 변경.
- **매물 개별 제어**: 웹 화면 상에서 직접 신규 매물을 추가하거나, 기존 매물의 정보 수정 및 삭제 처리.
- **엑셀 대량 갱신**: 신규 매물 엑셀 파일을 웹 UI에 직접 업로드하여 데이터베이스의 매물 데이터를 초기화하거나 대량 추가/갱신 가능.

## ⚙️ 로컬 실행 방법 (Installation & Run)

### 1. 패키지 설치
```bash
npm install
```

### 2. 초기 데이터 이관 (마이그레이션) 및 실행
처음 실행 시 엑셀 파일 내의 데이터가 SQLite 데이터베이스 파일(`database.sqlite`)로 자동 이관됩니다.
```bash
npm start
```
> 서버가 실행되면 [http://localhost:3000](http://localhost:3000)으로 접속하여 바로 이용하실 수 있습니다.

### 3. 초기 관리자 정보
- **아이디**: `admin`
- **비밀번호**: `adminpassword123`

## ☁️ 클라우드 퍼블리싱 (Render.com 배포 가이드)

본 소스 코드는 클라우드 환경 배포 시 데이터베이스와 엑셀 마이그레이션이 자동으로 수행되도록 세팅되어 있습니다.

1. [Render.com](https://render.com/)에 회원가입 및 로그인합니다.
2. **[New +] -> [Web Service]** 클릭 후 본 GitHub 리포지토리(`powermap`)를 연동합니다.
3. 설정 사항 입력:
   - **Runtime**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
4. 배포가 완료되면 Render에서 발급되는 고유 HTTPS 주소로 24시간 언제 어디서나 안전한 비공개 매물장 서비스를 무료로 운영하실 수 있습니다.
